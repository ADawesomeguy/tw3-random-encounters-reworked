Showcase

Add random encounters to The Witcher world. Bandits waiting to ambush you, or a pack of wolves attacking you. You can even start tracking large creatures wandering around you by following their tracks

Install

First install the two dependencies for the mod:

  • modBootstrap (the mod was tested with v0.4.1 but user reported it worked for v0.1)
  • modSharedImports

Then take the release directory (if you're feeling risky) or the release from nexusmods and drop its content (not the directory itself, the three sub-directories bin/mods/dlc) in your The Witcher 3 directory

Add the following line of code into The Witcher 3\Mods\modBootstrap-registry\content\scripts\local\mods_registry.ws 

add(modCreate_RandomEncountersReworked());
so the file looks like this : 
// ----------------------------------------------------------------------------
// ----------------------------------------------------------------------------
class CModRegistry extends CModFactory {
    protected function createMods() {
        // add mod creation calls here, like this:
        //
        // add(modCreate_());
        // ...

        // see example dir
        //add(modCreate_ExampleMod());
        //add(modCreate_ExampleEntityMod());
        //add(modCreate_UiExampleMod());

        add(modCreate_RandomEncountersReworked()); // <-- you added this line
    }
}
// ----------------------------------------------------------------------------
// ----------------------------------------------------------------------------

Start the game, let it compile the new scripts and open the new mod menu to configure the mod as you want (the default values are ridiculous + it need a loading screen to update the settings).

Done.
