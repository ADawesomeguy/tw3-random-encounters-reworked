Showcase

Add random encounters to The Witcher world. Bandits waiting to ambush you, or a pack of wolves attacking you. You can even start tracking large creatures wandering around you by following their tracks

Install

I made a video guide showing you how to install the mod for people who prefer videos over text youtube link

First install the four dependencies for the mod:

  • modBootstrap (the mod was tested with v0.4.1 but users reported it worked for v0.1)
  • modSharedImports
  • Community Patch - Base (you may not need it if you use Ghost Mode or W3EE because you already have it)

As of July 2021, nexusmods is no longer used for my mods. Updates will continue to be hosted on Github in order to retain full control over my projects and IP. That means you will have to download the recent versions on Github, an entirely free website for you that is also ad-free, respects your privacy and doesn't put any non-premium restrictions.

Then take the latest release from Github and drop its content (not the directory itself, the three sub-directories bin/mods/dlc) in your The Witcher 3 directory

Merge the changes introduced by the Shared Utils

Add the following line of code into The Witcher 3\Mods\modBootstrap-registry\content\scripts\local\mods_registry.ws 

add(modCreate_RandomEncountersReworked());
so the file looks like this : 
// ----------------------------------------------------------------------------
// ----------------------------------------------------------------------------
class CModRegistry extends CModFactory {
    protected function createMods() {
        // add mod creation calls here, like this:
        //
        // add(modCreate_());
        // ...

        // see example dir
        //add(modCreate_ExampleMod());
        //add(modCreate_ExampleEntityMod());
        //add(modCreate_UiExampleMod());

        add(modCreate_RandomEncountersReworked()); // <-- you added this line
    }
}
// ----------------------------------------------------------------------------
// ----------------------------------------------------------------------------

Start the game, let it compile the new scripts and open the new mod menu to configure the mod as you want (it needs a loading screen to update the values).

Done.
