Contracts

RER will randomly inject notices on the noticeboard. These notices will tell the mod to start one of the few contracts it offers when you pick them.

The mod currently offers three types of contracts:

  • Trails contracts
  • Nests contracts
  • Hordes contracts
And each one has its unique notice so that you can learn which notice leads to which type and pick only the ones you like.
